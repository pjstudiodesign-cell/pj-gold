# pj-gold
⚜️ Sistema de gestão de elite para o PJ Studio Design. Controle de orçamentos em PDF, fluxo de caixa e gestão de projetos com Python e Streamlit.
⚜️ PJ GOLD - Gestão de Elite para Studio de Design
O PJ GOLD é um ecossistema de gestão desenvolvido por Paulo Jovane (PJ Studio Design) para transformar a administração de serviços de design e software em uma experiência de alta performance. O sistema foca em organização, profissionalismo e agilidade na entrega de orçamentos e contratos.

🚀 Funcionalidades Principais
Painel de Controle Real: Visualização imediata de contratos fechados, faturamento disponível e contas a receber.

Gerador de Orçamentos PDF: Criação de documentos profissionais com cabeçalho completo da empresa, incluindo endereço, e-mail e contatos.

Gestão de Jobs: Acompanhamento em tempo real do status de produção e do status financeiro de cada projeto.

Backup de Segurança: Exportação completa de dados para Excel para garantir que as informações financeiras estejam sempre protegidas.

Configurações Personalizadas: Interface para ajuste rápido de dados comerciais e identidade do Studio.

🛠️ Tecnologias Utilizadas
Python: Linguagem base para toda a lógica de negócio.

Streamlit: Framework utilizado para criar uma interface web moderna e responsiva.

SQLite3: Banco de dados robusto para armazenamento local de projetos e configurações.

FPDF: Biblioteca para geração automatizada de orçamentos em formato PDF.

Pandas & XlsxWriter: Ferramentas de elite para manipulação de dados e geração de backups em Excel.

📂 Estrutura do Projeto
main.py: O coração do sistema com toda a interface e lógica.

pj_gold_data.db: Banco de dados relacional contendo o histórico de clientes e jobs.

requirements.txt: Arquivo de dependências para instalação rápida em novos ambientes ou na nuvem.

PDFs_Orcamentos/: Pasta organizada para armazenamento automático das cópias de orçamentos gerados.
